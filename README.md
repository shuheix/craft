## 概要
  工学（建築・機械）向けのQAアプリです。  
  各社ノウハウが多く、なかなか技術交流も少ない業界で、もっと技術の交流ができたらいいとおもい作成しました。
  
## 使用技術
  バックエンド
    Ruby v2.7.2
    Rails v6.1.4
    Rubocop
    Rspec
  フロントエンド
    React v17.0.2
    Typescript v4.1
    ESlint
    Prettier
    Cypress
    SWR
    ReactHookForm
  インフラ・その他
    FirebaseAuthentication
    Docker/Docker-compose
    CircleCI
    MySQL
    AWS
